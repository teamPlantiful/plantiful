# 🌱 Plantiful

> **3D 시각화를 이용한 반려식물 케어 서비스**

---

## 📆 프로젝트 기간

> 2025.10.30 ~ 2025.11.26

## 💻 과정명

> 구름 DEEP DIVE 프론트엔드 5회차

## 👥 팀 구성

| 이름   | 역할                                                         |
| ------ | ------------------------------------------------------------ |
| 최원형 | 검색·정렬 / 식물 상세 모달페이지                             |
| 박나현 | 디자인 시스템/ 공통 컴포넌트/ 식물 등록 페이지/ 식물 3D 박스 |
| 서민욱 | 레이아웃·헤더, Auth                                          |
| 진현지 | 식물 검색 모달/ 외부 API 연동/ 무한스크롤                    |

---

## 🌿 프로젝트 소개

> 반려식물을 직접 키우고자 하지만 **관리 방법을 잘 모르는 사람들**을 위해, **식물의 상태를 3D 그래픽으로 시각화하고 관리 주기를 돕는 웹 서비스**입니다.

- 본인의 반려식물을 기록하고 환경에 맞게 관리할 수 있음
- 물주기·영양제·분갈이 주기 등 케어 루틴을 체계적으로 관리
- Three.js를 활용한 **직관적 3D 시각화** 제공
- Supabase를 통한 인증 및 데이터 관리

---

## 🎯 프로젝트 목표

| 목표               | 설명                                                       |
| ------------------ | ---------------------------------------------------------- |

| **케어 관리 기능** | 물주기, 영양제, 분갈이 등의 관리 주기를 설정하고 확인 가능 |
| **사용자 관리**    | Supabase Auth로 계정 관리 및 인증 플로우 구성              |
| **UI/UX 향상**     | 공통 디자인 시스템 기반의 일관된 UI와 반응형 레이아웃 구현 |

---

## 🌈 기대 효과

- **재미있고 직관적인 식물 관리 경험** 제공
- **시각적 상태 표현**을 통한 식물 케어 이해도 향상
- 식물별 **케어 주기와 정보**를 한 곳에서 확인 가능
- 물주기 등 **주기 관리 기능**으로 식물의 건강한 성장에 도움

## ⚙️ 주요 기능

### 🏠 메인 페이지

- 레이아웃 / 헤더 구성
- 오늘의 식물 3D 렌더링
- 검색 / 정렬 필터 (디바운스 적용)
- 무한 스크롤
- 식물 카드 리스트 (1xN & 2xN 배치)
- 물주기 버튼 컴포넌트
- 등록 FAB 버튼
- React Query 쿼리키 관리

### 🔐 로그인 & 회원가입

- Supabase Auth 이메일 회원가입/로그인
- OAuth (Google) 연동
- 로그인/회원가입 폼 UI 구성

### 👤 마이페이지

- 닉네임 및 비밀번호 변경 기능

### 🌼 식물 등록 모달

- 식물 검색 및 선택
- 종류별 API 연동
- 사용자 입력으로 직접 추가 가능
- 사진 업로드 기능
- 커스텀 스크롤 디자인

### 🍃 식물 상세 모달

- 상세 정보 UI 구성
- 식물 정보 수정
- 삭제 기능

### 🧩 공통 컴포넌트

- Card, Dropdown, Input, Modal, SearchBar, Button, FAB 등
- 팀 공통 디자인 시스템 토큰을 기반으로 스타일 통일
- 재사용 가능한 레이아웃/섹션 컴포넌트 구성

### 💫 기타 페이지

- 반응형 레이아웃
